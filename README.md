## 🚀 Onchain Analysis

Onchain Analysis is a production-ready data platform designed to ingest, process, and analyze blockchain data in real time.

It enables developers and analysts to track wallet activity, detect whale transactions, and uncover smart money movements across blockchain networks.

### ✨ Features
- Real-time blockchain data ingestion (WebSocket / RPC)
- Transaction processing and normalization
- Whale detection and alerting
- Modular analytics pipeline
- API service for querying on-chain insights
- Interactive dashboard for visualization

### 🧠 Use Cases
- Smart money tracking
- Whale monitoring
- DeFi analytics
- Risk and anomaly detection
- On-chain research and insights

### 🏗️ Architecture
Built with a scalable microservices architecture using:
- Python (data processing)
- FastAPI (API layer)
- PostgreSQL (storage)
- Redis/Kafka (stream processing)
- Docker (deployment)
